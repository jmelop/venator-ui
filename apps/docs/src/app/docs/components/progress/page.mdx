## Default

A progress bar at 60%.

  <Progress value={60} />
</Preview>

<CodeBlock className="mt-6"

## Sizes

Use the `size` prop to control the track height. The default size is `md`.

<Preview>
  <div className="flex flex-col gap-4 w-full">
    <div className="flex flex-col gap-1">
      <span className="text-xs text-neutral-500 dark:text-neutral-400">sm</span>
      <Progress value={60} size="sm" />
    </div>
    <div className="flex flex-col gap-1">
      <span className="text-xs text-neutral-500 dark:text-neutral-400">md</span>
      <Progress value={60} size="md" />
    </div>
  </div>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Progress value={60} size="sm" />
<Progress value={60} size="md" />`}
/>

## Full and empty

`value` is clamped between 0 and 100.

<Preview>
  <div className="flex flex-col gap-4 w-full">
    <Progress value={0} />
    <Progress value={100} />
  </div>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Progress value={0} />
<Progress value={100} />`}
/>
