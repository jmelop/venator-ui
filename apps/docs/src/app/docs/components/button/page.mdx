## Variants

Use the `variant` prop to change the visual style. The default variant is `primary`.

<Preview>
  <div className="flex gap-3 flex-wrap items-center">
    <Button variant="primary">Primary</Button>
    <Button variant="secondary">Secondary</Button>
    <Button variant="outline">Outline</Button>
    <Button variant="ghost">Ghost</Button>
  </div>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Button variant="primary">Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="outline">Outline</Button>
<Button variant="ghost">Ghost</Button>`}
/>

## Sizes

Use the `size` prop to control the button's padding and font size.

<Preview>
  <div className="flex gap-3 flex-wrap items-center">
    <Button size="sm">Small</Button>
    <Button size="md">Medium</Button>
    <Button size="lg">Large</Button>
  </div>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Button size="sm">Small</Button>
<Button size="md">Medium</Button>
<Button size="lg">Large</Button>`}
/>

## Loading state

Set `isLoading` to show a spinner and automatically disable the button.

<Preview>
  <div className="flex gap-3 flex-wrap items-center">
    <Button variant="primary" isLoading>Loading</Button>
  </div>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Button variant="primary" isLoading>Loading</Button>`}
/>

## Disabled

Pass the native `disabled` attribute to prevent interaction.

<Preview>
  <div className="flex gap-3 flex-wrap items-center">
    <Button variant="primary" disabled>Disabled</Button>
  </div>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Button variant="primary" disabled>Disabled</Button>`}
/>

## Full width

Set `fullWidth` to make the button span its container.

<Preview>
  <Button variant="primary" fullWidth>Full width</Button>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Button variant="primary" fullWidth>Full width</Button>`}
/>
