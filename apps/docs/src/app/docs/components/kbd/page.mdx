## Combinations

Compose multiple `Kbd` elements with plain text to represent keyboard shortcuts.

<Preview>
  <div className="flex flex-col gap-3">
    <div className="flex gap-1.5 items-center text-sm text-neutral-600 dark:text-neutral-400">
      <Kbd>⌘</Kbd>
      <span>+</span>
      <Kbd>K</Kbd>
    </div>
    <div className="flex gap-1.5 items-center text-sm text-neutral-600 dark:text-neutral-400">
      <Kbd>Ctrl</Kbd>
      <span>+</span>
      <Kbd>Shift</Kbd>
      <span>+</span>
      <Kbd>P</Kbd>
    </div>
  </div>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<Kbd>⌘</Kbd> + <Kbd>K</Kbd>

<Kbd>Ctrl</Kbd> + <Kbd>Shift</Kbd> + <Kbd>P</Kbd>`}
/>
