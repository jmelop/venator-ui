## Basic

`ModuleGrid` renders a responsive `grid` container. Children stack to a single column on mobile, expand to the specified column count on larger screens.

<Preview>
  <ModuleGrid>
    <Card>
      <CardHeader title="Total users" />
      <CardContent><p className="text-2xl font-bold text-neutral-900">4,821</p></CardContent>
    </Card>
    <Card>
      <CardHeader title="Active sessions" />
      <CardContent><p className="text-2xl font-bold text-neutral-900">312</p></CardContent>
    </Card>
    <Card>
      <CardHeader title="Revenue" />
      <CardContent><p className="text-2xl font-bold text-neutral-900">$18,400</p></CardContent>
    </Card>
  </ModuleGrid>
</Preview>

<CodeBlock className="mt-6"
  language="tsx"
  code={`<ModuleGrid>
  <Card>…</Card>
  <Card>…</Card>
  <Card>…</Card>
</ModuleGrid>`}
/>

## Columns

Use the `columns` prop to set the number of columns at the `lg` breakpoint. All values collapse to a single column on small screens and two columns on medium screens.

<CodeBlock className="mt-6"
  language="tsx"
  code={`{/* 1 column at all breakpoints */}
<ModuleGrid columns={1}>…</ModuleGrid>

{/* 2 columns at lg+ */}
<ModuleGrid columns={2}>…</ModuleGrid>

{/* 3 columns at lg+ (default) */}
<ModuleGrid columns={3}>…</ModuleGrid>

{/* 4 columns at lg+ */}
<ModuleGrid columns={4}>…</ModuleGrid>`}
/>

## Gap

Use the `gap` prop to control spacing between grid cells.

<CodeBlock className="mt-6"
  language="tsx"
  code={`{/* gap-3 */}
<ModuleGrid gap="sm">…</ModuleGrid>

{/* gap-5 (default) */}
<ModuleGrid gap="md">…</ModuleGrid>

{/* gap-7 */}
<ModuleGrid gap="lg">…</ModuleGrid>`}
/>
